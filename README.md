# portfolio_website

Project: Room & Mess Finder — Full-Stack Web Application
I built Room & Mess Finder, a full-stack web application designed to simplify property discovery across India. The platform helps users quickly find rooms and mess options with a modern, high-performance UX.

What I built
Real-time data powered by React Query for responsive, up-to-date experiences
Secure authentication using Supabase
Interactive location browsing with Leaflet maps
AI-powered chatbot using Google Gemini to assist users and answer queries
Payments integration with Razorpay
Polished UI using Tailwind CSS, plus smooth animations and user-friendly flows
Tech Stack
React, TypeScript, Vite, Tailwind CSS, Framer Motion, shadcn/ui, React Query, Supabase, Google Gemini AI, Razorpay, Leaflet Maps
